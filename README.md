# 📊 HR Analytics Dashboard — Power BI

## 🔍 Overview
An interactive HR Analytics dashboard built in Power BI to analyze 
employee attrition, satisfaction, and workforce demographics.

## 📌 Key Metrics
- **Total Employees:** 1,470
- **Attrition Count:** 237
- **Attrition Rate:** 16.12%
- **Active Employees:** 1,233
- **Average Age:** 37

## 📈 Visuals & Features
- KPI Cards for quick summary
- Department-wise attrition (Pie Chart)
- Age group distribution (Bar Chart)
- Job Satisfaction Rating Matrix
- Education field attrition (Bar Chart)
- Attrition by gender across age groups (Donut Charts)
- Interactive slicers by Education Level

## 💡 Key Insights
- R&D department has the highest attrition (~56%)
- Employees aged 25–34 form the largest workforce segment
- Life Sciences education field shows highest attrition count
- Attrition rate is highest in the Under 25 age group (52.63%)

## 🛠️ Tools Used
- Microsoft Power BI Desktop
- Microsoft Excel / CSV (data source)
- DAX (for calculated measures)
